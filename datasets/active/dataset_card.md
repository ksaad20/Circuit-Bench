# Active Components Benchmark Dataset

## Overview

The Active Components Dataset focuses on circuits containing semiconductor devices used in modern electronic systems.

## Objectives

- Device modelling
- Parameter prediction
- Performance estimation
- AI-assisted design

## Included Components

- Bipolar Junction Transistors
- MOSFETs
- JFETs
- Operational Amplifiers
- Voltage Regulators

## Benchmark Tasks

- Device Parameter Estimation
- Performance Prediction
- Fault Detection
- Surrogate Modelling

## Inputs

- Device parameters
- Bias conditions
- Circuit topology

## Outputs

- Voltages
- Currents
- Gain
- Power

## Simulator Support

- ngspice
- PySpice

## Evaluation Metrics

- MAE
- RMSE
- Runtime
