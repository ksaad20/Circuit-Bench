# Circuit-Bench Official Benchmark Results

## Purpose

This directory contains the official benchmark results for Circuit-Bench.

All published benchmark results should follow the official evaluation protocol and use versioned benchmark datasets.

---

## Directory Structure

baseline_results/
benchmark_reports/

---

## Reproducibility

Each benchmark result should record:

- Circuit-Bench version
- Dataset version
- Benchmark task
- Model
- Hyperparameters
- Random seed
- Python version
- Hardware
- Runtime
- Evaluation metrics
